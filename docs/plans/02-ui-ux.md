# UI/UX — Revolut 브랜드 적용

## 브랜드 결정
- **Primary**: Revolut. 핀테크 톤 + 학생용으로 친근한 cobalt violet은 학습 도구로도 적절.
- **Sub palette**: 카테고리 칩에 Revolut의 accent 팔레트 (teal, orange, pink, light-green 등)를 illustration 용도로만 사용.

## 디자인 토큰 (Revolut 매핑)
- `--brand`: `#494fdf` (primary)
- `--brand-bright`: `#4f55f1`
- `--ink`: `#191c1f`
- `--canvas-dark`: `#000000`
- `--canvas-light`: `#ffffff`
- `--surface-soft`: `#f4f4f4`
- `--surface-elevated`: `#16181a`
- `--hairline-light`: `#e2e2e7`
- `--accent-teal`: `#00a87e` (잔액 +)
- `--accent-orange`: `#ec7e00` (지출)
- `--accent-pink`: `#e61e49` (잔액 -)
- `--accent-light-green`: `#428619`

### 카테고리 6종 색 (도넛 차트 + 칩)
- 학용품 — `#494fdf` (cobalt violet)
- 간식 — `#ec7e00` (warning orange)
- 교통 — `#007bc2` (light blue)
- 저축 — `#00a87e` (teal)
- 선물 — `#e61e49` (pink)
- 기타 — `#8d969e` (stone)

## 타이포
- Display: 시스템 폰트 스택 (`-apple-system, "Segoe UI", "Noto Sans KR", sans-serif`) — Aeonik Pro 라이선스 없음. weight 600~700, line-height 1.0, letter-spacing -0.02em.
- Body: 같은 스택, weight 400/600.
- 클램프 크기: 헤드라인 `clamp(40px, 7vw, 80px)`.

## 접근성 체크리스트
- 모든 텍스트 대비 4.5:1 이상 (검정 배경에 흰색, 흰 배경에 ink).
- 입력 폼에 `<label>` + `aria-describedby`.
- 삭제 버튼 `aria-label="OO 거래 삭제"`.
- 도넛/바 차트에 `aria-label` 요약 텍스트 동반.
- 키보드 탭 순서: 폼 → 거래 리스트 → 차트 → 내보내기.
- focus-visible outline: 2px solid `--brand-bright`.
- 색 단독 의존 금지 — 카테고리 칩은 색 + 이모지 + 텍스트.

## 카피 (한국어)
- 헤드라인: "이번 달 내 용돈" / "용돈 흐름을 한 눈에 보기"
- 빈 상태: "아직 기록이 없어요. 첫 거래를 입력해 보세요!"
- 잔액 음수 경고: "이번 달 지출이 수입보다 많아요. 다음 달에 균형을 맞춰 봐요."
- 푸터: "기록은 이 PC에만 저장돼요. JSON 내보내기로 백업하세요."

## 반응형
- ≥1024px: 입력 폼·리스트 2열, 차트 2열.
- 768~1023px: 모두 1열, 카드 폭 60ch.
- ≤767px: 헤드라인 클램프 다운, 모든 카드 풀 폭.
